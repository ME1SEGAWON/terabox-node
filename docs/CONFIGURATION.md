## Configuration

1. Open your Terabox cloud.
2. Open the browser's developer tools (F12).
3. Go to the "Application" tab.
4. Select the "Cookies" item in the left panel.
5. Look for the "ndus" cookie value and copy it to ".config.yaml".

```yaml
accounts:
  MyMainAcc: Y-DSDSD...
  MySecondAcc: YDvrwD...
```
